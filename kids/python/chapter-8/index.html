字符串特有的相关操作

  1. 转义字符

    为了输出一些特殊的字符，比如引号、缩进、换行等，在大部分编程语言的字符串中都有转义字符。

    转义字符由反斜杠 \ 开头

    常用的转义字符如下：

    转义符 描述
    \ 反斜杠（\）
    ' 单引号（'）
    " 双引号（"）
    \n 换行符（LF）
    \r 回车符（CR）
    \t 水平制表符（TAB）

  2. 格式化输出

    • %格式化输出

      符号 描述
      %s 格式化整数或字符串，如果s前面有数字，则代表所要站的位置长度
      %f 格式化浮点数，如果f前面有数字，则代表保留的小数精度

    • format格式化输出

      
>>>print('1乘以{0}等于{0}，2乘以{1}等于{2}。'.format(3,4,8))


>>>print('我叫{name}, 这里是我的{program}课。'.format(name='kwazii',program='python'))

  3. split()

    将一个字符串使用分割符分割成字符串列表。

    
>>>fruit = "苹果,香蕉,橘子,葡萄"
>>>fruit.split(",")

['苹果','香蕉','橘子','葡萄']

  4. join()

    和split()的功能相反

    
>>>fruit = ['苹果','香蕉','橘子','葡萄']
>>>",".join(fruit)

"苹果,香蕉,橘子,葡萄"

总结回顾

字符串是我们使用最多的数据类型，它的一些常用操作在程序中的使用频次也是最高的。字符串作为序列的一种，它拥有序列的一些通用操作。同时字符串作为程序最重要的输入输出手段，格式化输出一个字符串也是非常重要的。另外不要忘记转义字符，在某些情况下它们特别有用。
